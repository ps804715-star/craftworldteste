\# 🎮 CW Game Tracker Pro



Aplicação profissional para gestão industrial e análise financeira do CW Game.



\## 🚀 Funcionalidades



\- 📊 \*\*Preços em Tempo Real\*\* - GeckoTerminal API

\- 🏭 \*\*Gestão de Fábricas\*\* - Níveis, Workers, Workshop, Mastery

\- 💰 \*\*Calculadora de Lucros\*\* - ROI, Taxas, Price Impact

\- 📈 \*\*Gráficos de Evolução\*\* - Histórico de preços

\- 🤖 \*\*AI Trend Analysis\*\* - Tendências e recomendações



\## 🛠️ Instalação



```bash

\# Instalar dependências

npm install



\# Desenvolvimento

npm run dev



\# Build para produção

npm run build



\# Preview

npm run preview
